# AgroVision - UI/UX Design Document

## Design Philosophy

**Visual Identity**: Modern, clean, and agricultural-themed with green color palette representing growth, sustainability, and nature.

**Accessibility**: Simple navigation for rural farmers with varying tech literacy levels.

**Offline-First**: Critical features work without internet connection.

## Color Palette

### Primary Colors
- **Primary Green**: `#2D6A4F` - Headers, primary buttons, active states
- **Secondary Green**: `#52B788` - Secondary actions, highlights
- **Accent Green**: `#74C69D` - Accents, icons
- **Light Green**: `#B7E4C7` - Backgrounds, cards
- **Pale Green**: `#D8F3DC` - Very light backgrounds

### Semantic Colors
- **Success**: `#06D6A0` - Success messages, growth indicators
- **Warning**: `#F77F00` - Warnings, alerts
- **Error**: `#D00000` - Errors, critical alerts
- **Info**: `#0077B6` - Information, tips

### Typography
- **Font Family**: Inter (clean, modern, readable)
- **Heading 1**: 32px, Bold
- **Heading 2**: 24px, SemiBold
- **Heading 3**: 20px, SemiBold
- **Body**: 16px, Regular
- **Caption**: 14px, Regular
- **Small**: 12px, Regular

## Screen-by-Screen Design

### 1. Splash Screen
**Purpose**: Initial loading, app branding

**Layout**:
- Centered logo with plant/growth animation
- App name "AgroVision" below logo
- Tagline: "Smart Farming for Rwanda"
- Loading indicator at bottom

**Animation**: 
- Logo fades in with upward motion
- Plant grows from seed to sprout (2 seconds)

---

### 2. Onboarding (3 Screens)

**Screen 1: AI Diagnosis**
- Hero Image: Farmer scanning crop with phone
- Headline: "Detect Crop Diseases Instantly"
- Description: "Take a photo and get instant diagnosis with treatment recommendations"

**Screen 2: Market Prices**
- Hero Image: Market price chart with trending crops
- Headline: "Know the Best Prices"
- Description: "Real-time market prices from all major markets in Rwanda"

**Screen 3: Farm Management**
- Hero Image: Farm dashboard with crops
- Headline: "Manage Your Farms Digitally"
- Description: "Track planting, fertilizer, harvests, and costs in one place"

**Navigation**: Dots indicator, Skip button, Get Started button

## Animations & Transitions

1. **Page Transitions**: Slide from right (300ms)
2. **Card Tap**: Scale down to 0.95 with shadow
3. **Loading States**: Shimmer effect on cards
4. **Pull to Refresh**: Custom plant growing animation
5. **Success Actions**: Checkmark with green ripple
6. **Error States**: Shake animation with red highlight

---

## Accessibility Features

1. **Font Scaling**: Support system font size
2. **High Contrast**: Option for high contrast mode
3. **Screen Reader**: All elements properly labeled
4. **Touch Targets**: Minimum 48x48dp
5. **Color Blindness**: Don't rely solely on color
6. **Offline Indicators**: Clear offline/online status

---

## Responsive Design

- **Small Phones** (< 360dp): Single column, compact spacing
- **Standard Phones** (360-420dp): Comfortable spacing
- **Large Phones/Small Tablets** (> 420dp): Utilize extra space for side-by-side layouts

---

## Design Assets Needed

### Icons (Custom SVG)
- Crop types (maize, beans, potato, etc.)
- Weather conditions
- Farm activities
- Disease types
- Market categories

### Illustrations
- Onboarding screens (3 custom illustrations)
- Empty states (no farms, no listings, etc.)
- Success/error states
- Loading animations

### Images
- Placeholder crop images
- Disease examples (for reference)
- Tutorial screenshots

---

## Prototyping Tools Recommended

- **Figma** - For complete UI design and prototyping
- **Flinto** - For advanced animations
- **Principle** - For micro-interactions

## Design System Export

Create a design system with:
- All color tokens
- Typography scale
- Component library
- Icon set
- Animation guidelines

This can be exported from Figma to Flutter using tools like **Figma to Flutter** or manually implementing with the theme system.
